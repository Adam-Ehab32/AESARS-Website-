Quick Reference — STM32F401 & ARMCLANG

    


      

        
🔌
GPIO MODER Register

        
2-bit mode field per pin (RM0368 §8.4.1). Offset +0x00 from GPIO base address.

        

          
00Input mode (reset state)

          
01General purpose output

          
10Alternate function mode

          
11Analog mode

        

        

          GPIOA: 0x40020000
          GPIOB: 0x40020400
          GPIOC: 0x40020800
        

      


      

        
📤
OTYPER / OSPEEDR

        
Output type (§8.4.2) and speed (§8.4.3). Offsets +0x04 and +0x08.

        

          
OTYPER=0Push-pull output

          
OTYPER=1Open-drain output

          
OSPEEDR=00Low speed 2 MHz

          
OSPEEDR=01Medium 25 MHz

          
OSPEEDR=10High speed 50 MHz

          
OSPEEDR=11Very high 100 MHz

        

      


      

        
⬆️
PUPDR Register

        
Pull-up / pull-down configuration (RM0368 §8.4.4). Offset +0x0C from GPIO base.

        

          
00No pull-up / pull-down

          
01Pull-up enabled

          
10Pull-down enabled

          
11Reserved

        

      


      

        
🕐
RCC Clock Enable

        
Enable peripheral clocks before access (RM0368 §6). AHB1/APB1/APB2 buses.

        

          
AHB1ENR0x40023830 · GPIO A–E

          
APB1ENR0x40023840 · TIM2–5, USART2–3

          
APB2ENR0x40023844 · TIM1, USART1, SPI1

          
GPIOAENBit 0 of AHB1ENR

          
GPIOBENBit 1 of AHB1ENR

          
TIM2ENBit 0 of APB1ENR

        

      


      

        
🔤
ARMCLANG Syntax Rules

        
Keil MDK armasm unified syntax conventions for Cortex-M4 (THUMB ISA).

        

          
LabelsColumn 0, zero indent

          
InstructionsTAB-indented

          
PROC/ENDPRequired function wrappers

          
MOVW/MOVTSplit 32-bit immediates

          
THUMBISA directive required

          
EXPORTExpose symbol globally

          
ALIGNRequired after literal pools
